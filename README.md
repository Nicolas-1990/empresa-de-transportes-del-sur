# Empresa de Transportes del Sur

Sitio web responsive desarrollado con HTML y CSS puro.

## Características

- Diseño responsive
- Menú mobile
- Navegación entre líneas y recorridos
- Organización por páginas
- Integración con Google Maps

## Tecnologías utilizadas

- HTML5
- CSS3
- Boxicons
- Remixicon
